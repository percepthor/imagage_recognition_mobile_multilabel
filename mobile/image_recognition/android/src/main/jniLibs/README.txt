Place your native libraries (.so files) here:

- arm64-v8a/libimage_recognition.so  (for 64-bit ARM devices)
- armeabi-v7a/libimage_recognition.so  (for 32-bit ARM devices)
- x86_64/libimage_recognition.so  (for x86 64-bit emulators/devices)
- x86/libimage_recognition.so  (for x86 32-bit emulators/devices)

These directories are configured in build.gradle to be packaged with the app.
